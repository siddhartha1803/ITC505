Addendum:
The Smart Budget Tracker models the complete responsive Holy Grail Layout that is built on CSS Grid. Every structural component is fluidly scaled at any screen size: header, sidebar, main, advertisement and footer. The project uses JavaScript authorativeness in form of a live cost tracker. The program reacts dynamically to event-driven DOM manipulation to update the list, total and average values as users add a single item and an amount. To do this, it was necessary to prevent the default way of submitting the form with preventDefault and process the data with array functions such as push and reduce. Problems with grid cell resizing occurred early in the development stage with grid cells often resizing unevenly; the introduction of minmax() and auto-fit fixed responsiveness. The form-validation edge cases were debugged to have correct numeric parsing, and the CSS additions that gave such features as transparent panels and hover effect to the design gave the overall professional appearance. In general, this assignment provided me with a better insight into the relationship between structural layout, event handling, and dynamic rendering to create interactive web applications.
