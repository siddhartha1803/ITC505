Lab 3: Serving Fonts

For this assignment, I selected the “Nosifer” font, which is not normally available on most devices. By linking it from Google Fonts, the browser downloads the font when the page is loaded.

The process works like this:

  1. A link to the font required must be added in the <head> of the HTML file.
  2. The browser downloads the font file on Google.
  3. CSS font-family and font-family to elements can be used to define the font.
  4. When the page is loaded, any text formatted with that family is custom formatted with the custom font.
